# ADR-B-002 — Official Metaphor: Mapa Estratégico do Contexto

**Marca:** EcoSeg Seguros

**Identificador:** ADR-B-002

**Categoria:** Brand Architecture Decision Record

**Status:** Accepted

**Versão:** 1.0

## Decision

A metáfora oficial é o Mapa Estratégico do Contexto.

## Context

This formalizes, as a standalone citable decision, what `BD-003 §4` already established and `DES-004`'s Nota de Governança already adopted by direct decision (in lieu of the empirical validation `VIS-001.3` never performed). No new content — this ADR exists so the metaphor has its own decision record, separate from the documents that merely use it.

## Consequences

Every future brand/visual/experience artifact must be checkable against this single metaphor. Any proposal that cannot be described as "núcleo + camadas de contexto" fails by default (`VIS-001 §1.4` criteria, `DES-006`'s Teste de Reconhecimento).

## Traceability

Formalizes: `BD-003 §4` (Strategic Context Map), `VIS-001 §VIS-001.3A.3` (fundamental-unit hypothesis, adopted by decision in `DES-004`).
