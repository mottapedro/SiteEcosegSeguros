# ADR-B-004 — Website Communicates a Journey, Not a Catalog

**Marca:** EcoSeg Seguros

**Identificador:** ADR-B-004

**Categoria:** Brand Architecture Decision Record

**Status:** Accepted

**Versão:** 1.0

---

## Decision

O website comunica uma jornada, não um catálogo de produtos.

## Context

Formalizes as a standalone decision record what `DES-002` (Information Architecture) and `DES-007` (Home Narrative Design) already established in practice: sections exist because they represent a stage of the Progress Protection Philosophy (`BD-003 §6`), not because they are conventional "site institucional" pages. `DES-007`'s own Reconciliação section already identified where the live implementation still violates this — Products appearing before Adversidades/Compreensão in the current page order.

## Consequences

This ADR does not change anything not already specified in `DES-002`/`DES-007` — it makes the underlying principle citable independently, so future pages (Soluções, Segmentos, Metodologia — none built yet) can be checked against it without needing to re-derive the reasoning from the full IA document.

## Traceability

Formalizes: `DES-002` (Information Architecture), `DES-007` (Home Narrative Design, especially its Reconciliação section).

Affects: any future page built under `WEB-002`/`WEB-003`/`WEB-004`.
